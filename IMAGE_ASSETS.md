## How to Integrate

1. Place all `.webp` files in `src/assets/`
2. Import at the top of each page file, e.g.:
   ```js
   import heroBanner from '../assets/hero-banner.webp';
   ```
3. Replace the Google CDN URL string with the imported variable, e.g.:
   ```jsx
   // Before:
   style={{ backgroundImage: "url('https://lh3.googleusercontent.com/...')" }}
   // After:
   style={{ backgroundImage: `url('${heroBanner}')` }}
   ```
4. For `<img>` tags:
   ```jsx
   // Before:
   <img src="https://lh3.googleusercontent.com/..." />
   // After:
   <img src={combedCottonImg} />
   ```
